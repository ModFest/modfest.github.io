ModFest 1.15 Mod Pack

The ModFest 1.15 Mod Pack contains all 39 of the ModFest 1.15 entries so that players can experiment with them before they vote in the award polls.

Copy Pack Download URL

How To Install:

  1. Download & Install MultiMC.
  2. In MultiMC, click on "Add Instance"
  3. Select the "Import from zip" option from the list on the left.
  4. Click the link below to copy it, go back to MultiMC, and paste it into the text box labeled "Local file or link to a direct download".
    5. https://legacy.modfest.net/1.15/pack/modfest.zip
  5. Click "Launch" once to initialize auto-updating, and then it will crash saying "The process failed to start."
  6. Click "Launch" again to run the pack.
  7. You should see a dialog appear that has a big "Install" button on it. Click on the Install button.
  8. When you launch the game, the ModFest official server will be in the Multiplayer tab.

Additional Mods Included:

  • CustomTitleScreen
  • HWYLA
  • Lithium
  • MiniHud
  • Mod Menu
  • ModFest Extras
  • Patchouli
  • Roughly Enough Items
  • Tech Reborn
  • Trinkets

Note: This pack only works with MultiMC, due to many mods not being available on CurseForge.
