ModFest 1.17 Pack

The ModFest 1.17 Pack contains all 32 of the ModFest 1.17 entries so that players can experiment with them before they vote in the award polls.

disclaimer: Hyperstellar has had to be removed from the pack due to crashes preventing the server from running. Please try it out on your own with this link!

Copy Pack Download URL

How to install:

  1. Download & Install MultiMC.
  2. In MultiMC, click on "Add Instance"
  3. Select the "Import from zip" option from the list on the left.
  4. Click the link below to copy it, go back to MultiMC, and paste it into the text box labeled "Local file or link to a direct download".
    5. https://modfest.net/1.17/pack/modfest-1-17.zip
  5. Click "Launch" once to initialize auto-updating, and then it will crash saying "The process failed to start."
  6. Click "Launch" again to run the pack.
  7. You should see a dialog appear that has a big "Install" button on it. Click on the Install button.
  8. After installation, the pack should launch. The ModFest Server should already be listed in the Multiplayer menu, but if not, the address is play.modfest.net (or play.dblsaiko.net:5013 if the former doesn't work)
