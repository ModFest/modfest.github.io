Thanks for participating in MiniFest: FallFest 1.16!

FallFest 1.16 ran from Thursday, October 1st thru Thursday, October 15th, 2020. It was a big success and over 30 mods were submitted!

If you are interested in visiting the server, you can download the pack here.

How to install:

  1. Download & Install MultiMC.
  2. In MultiMC, click on "Add Instance"
  3. Select the "Import from zip" option from the list on the left.
  4. Click the link above to copy it, go back to MultiMC, and paste it into the text box labeled "Local file or link to a direct download".
    5. https://modfest.net/fallfest/1.16/server/fallfest-1.3.0.zip
  5. Click "Launch" once to initialize auto-updating, and then it will crash saying "The process failed to start."
  6. Click "Launch" again to run the pack.
  7. You should see a dialog appear that has a big "Install" button on it. Click on the Install button.
  8. After installation, the pack should launch. The ModFest Server should already be listed in the Multiplayer menu, but if not, the address is play.modfest.net

Once you are in-game, mess around in Source3 tutorial world, then head to multiplayer and join the FallFest server.

The server rules are simple: don't cause problems!

We hope you enjoy exploring the map as much as we enjoyed creating it.
